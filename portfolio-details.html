Brain Tumor Detection

This project focuses on classifying types of brain tumors using **Deep Learning and Numerical Tensor Methods**. The model leverages synthetic data generated from a forward geophysical model and applies Gaussian Processes for parameter estimation and training the machine learning algorithm. It addresses a non-linear, ill-posed inverse problem, enabling accurate and efficient tumor classification.

Technologies Used:

  • Python, TensorFlow, PyTorch
  • Numerical Tensor Methods
  • Gaussian Processes
  • Data Visualization & Preprocessing

Key Outcomes:

  • High-accuracy classification of brain tumor types
  • Robust parameter estimation using GP-based methods
  • Improved training efficiency for deep learning models
